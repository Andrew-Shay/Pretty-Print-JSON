# Pretty Print JSON
Command line tool for printing pretty JSON

# Usage

#### Help
`ppj -h`

#### Pretty Print JSON
`ppj p FILE_NAME`

#### Overwrite JSON file
`ppj p FILE_NAME -o`

#### Write new file
`ppj p FILE_NAME -f NEW_FILE_NAME`

# Install
`pip install pretty_print_json`

# Requirements
Python 3.5 or 2.7
PIP
