# Pretty Print JSON
Command line tool for printing pretty JSON

# Usage

#### Help
`ppj -h`

#### Pretty Print JSON
`ppj FILE_PATH`

#### Overwrite JSON file
`ppj FILE_PATH -o`

#### Write new file
`ppj FILE_PATH -f NEW_FILE_PATH`

# Install
`pip install pretty_print_json`

# Requirements
Python 3.5 or 2.7
